# 命题 1：作等边三角形

## 原题陈述

**命题：** 在已知有限直线上作一个等边三角形。

**已知：** 线段 AB

**求作：** 在 AB 上作等边三角形

---

## 我的证明

### 构造步骤

```
1. 以 A 为圆心，AB 为半径作圆
2. 以 B 为圆心，BA 为半径作圆
3. 两圆相交于点 C
4. 连接 AC、BC
```

### 证明

**断言：** △ABC 是等边三角形

**证明过程：**

1. **AC = AB**（根据构造步骤1，C 在以 A 为圆心、AB 为半径的圆上）
   - 圆的定义：圆上所有点到圆心的距离相等

2. **BC = BA**（根据构造步骤2，C 在以 B 为圆心、BA 为半径的圆上）
   - 同理，圆的定义

3. **AB = BA**（同一线段，自反性）

4. 由公理1（等于同量的量彼此相等）：
   - AC = AB = BC

5. **因此，AC = BC = AB**

**结论：** △ABC 三边相等，是等边三角形。

## 验证

让我验证这个构造的正确性：

```python
# 用坐标几何验证
import math

# 设 A = (0, 0), B = (1, 0)
A = (0, 0)
B = (1, 0)

# 两圆交点计算
# 圆A: x² + y² = 1
# 圆B: (x-1)² + y² = 1
# 解得: x = 1/2, y = ±√3/2

C1 = (0.5, math.sqrt(3)/2)
C2 = (0.5, -math.sqrt(3)/2)

def distance(p1, p2):
    return math.sqrt((p1[0]-p2[0])**2 + (p1[1]-p2[1])**2)

# 验证三边相等
AB = distance(A, B)
AC = distance(A, C1)
BC = distance(B, C1)

print(f"AB = {AB:.6f}")
print(f"AC = {AC:.6f}")
print(f"BC = {BC:.6f}")
print(f"相等？{abs(AB-AC) < 1e-10 and abs(AB-BC) < 1e-10}")
```

**输出：**
```
AB = 1.000000
AC = 1.000000
BC = 1.000000
相等？True
```
