# Prop 3: 截取较大线段等于较小线段

**命题：** 给定两条不等长线段，从较大线段截取一段等于较小线段。

**已知：** AB > C
**求作：** 在AB上截取AD = C

**构造：**
1. 从A作AE = C（用Prop 2）
2. 以A为圆心，AE为半径作圆
3. 圆交AB于D
4. AD = AE = C

**证毕。**

# Prop 4: 边角边全等 (SAS)

**命题：** 两三角形两边及其夹角对应相等，则全等。

**已知：** AB=DE, AC=DF, ∠A=∠D
**求证：** △ABC ≅ △DEF

**证明：**
1. 若BC ≠ EF，将△ABC叠到△DEF上
2. A与D重合，AB沿DE，AC沿DF
3. B与E重合（AB=DE）
4. C与F重合（AC=DF）
5. 因此BC与EF重合
6. 两三角形完全重合，全等

**证毕。**

# Prop 5: 等腰三角形底角相等

**命题：** 等腰三角形两底角相等。

**已知：** AB = AC
**求证：** ∠ABC = ∠ACB

**证明：**
1. 延长AB到D，AC到E，使BD = CE（Prop 3）
2. 连接BE、CD
3. △ABE ≅ △ACD (SAS: AB=AC, ∠A公共, AE=AD)
4. 因此BE = CD，∠ABE = ∠ACD，∠AEB = ∠ADC
5. 又BD = CE，故△BDC ≅ △CEB (SAS)
6. 因此∠DBC = ∠ECB
7. ∠ABC = ∠ABE - ∠DBC = ∠ACD - ∠ECB = ∠ACB

**证毕。**
（史称"驴桥"定理，初学者之难关）

# Prop 6: 等腰三角形逆定理

**命题：** 两角相等的三角形是等腰三角形。

**已知：** ∠B = ∠C
**求证：** AB = AC

**反证法：**
1. 设AB ≠ AC，不妨设AB > AC
2. 在AB上截取AD = AC（Prop 3）
3. 连接DC
4. △ADC等腰，故∠ADC = ∠ACD（Prop 5）
5. 但∠ADC > ∠B（外角），∠ACD < ∠ACB = ∠B
6. 矛盾！因此AB = AC

**证毕。**

# Prop 7: 同底同侧等边三角形必重合

**命题：** 同一直线同侧不能作两个不同的等边三角形。

**反证法：**
1. 设同侧有△ABC和△ABD，都等边
2. 则AC = BC = AB = AD = BD
3. 在△ACD中，AC = AD，故∠ACD = ∠ADC
4. 在△BCD中，BC = BD，故∠BCD = ∠BDC
5. 因此∠ACB = ∠ACD + ∠BCD = ∠ADB
6. 但△ABC和△ABD都是等边，∠ACB = ∠ADB = 60°
7. 两三角形完全重合

**证毕。**

# Prop 8: 边边边全等 (SSS)

**命题：** 三边对应相等，则三角形全等。

**已知：** AB=DE, BC=EF, CA=FD
**求证：** △ABC ≅ △DEF

**证明：**
1. 将△ABC叠到△DEF上，使B与E重合，BC沿EF
2. C与F重合（BC=EF）
3. 若A不在D处，设落在G
4. 则EG = AB = ED，FG = AC = FD
5. △EDG和△FDG都等腰
6. 但同底EF同侧不能有两不同等腰三角形（Prop 7）
7. 因此G与D重合，两三角形全等

**证毕。**

# Prop 9: 平分已知角

**命题：** 平分一个已知角。

**已知：** ∠BAC
**求作：** 平分线

**构造：**
1. 在AB上任取D
2. 在AC上截取AE = AD（Prop 3）
3. 连接DE
4. 在DE上作等边三角形△DEF（Prop 1，F在角内）
5. 连接AF
6. AF平分∠BAC

**证明：**
1. AD = AE（构造）
2. DF = EF（等边三角形）
3. AF公共
4. △ADF ≅ △AEF (SSS)
5. 因此∠DAF = ∠EAF

**证毕。**

# Prop 10: 平分已知线段

**命题：** 平分一个已知线段。

**已知：** AB
**求作：** 中点

**构造：**
1. 在AB上作等边三角形△ABC（Prop 1）
2. 作∠ACB的平分线CD，交AB于D
3. D为AB中点

**证明：**
1. CA = CB（等边三角形）
2. CD公共
3. ∠ACD = ∠BCD（角平分线）
4. △ACD ≅ △BCD (SAS)
5. 因此AD = BD

**证毕。**
