# Prop 11: 过直线上一点作垂线

**命题：** 过已知直线上一点作垂直于该直线的直线。

**已知：** 直线AB，点C在AB上
**求作：** 过C垂直于AB的直线

**构造：**
1. 在AB上取D使CD = CE（任取等长线段）
2. 在DE上作等边三角形△DFE（Prop 1）
3. 连接FC
4. FC ⊥ AB

**证明：**
1. DC = EC（构造）
2. DF = EF（等边三角形）
3. FC公共
4. △DCF ≅ △ECF (SSS)
5. ∠DCF = ∠ECF，且和为180°，故各为90°

**证毕。**

# Prop 12: 过直线外一点作垂线

**求作：** 过C垂直于AB的直线

2. 以C为圆心，CD为半径作圆，交AB于E、F

# Prop 13: 邻补角和为180°

**命题：** 一直线与另一直线相交，邻补角之和等于两直角。

**已知：** 直线AB与CD交于B
**求证：** ∠ABC + ∠ABD = 180°

**证明：**
1. 若∠ABC = ∠ABD，则各为90°（定义直角）
2. 若不等，作BE ⊥ CD（Prop 11）
3. ∠ABC = ∠ABE + ∠EBC
4. ∠ABD = ∠ABE - ∠DBE = ∠ABE - ∠EBC（因∠EBC = ∠DBE，对顶角...修正：此处需调整）

**更简证法：**
1. 过B作BF ⊥ AB（Prop 11）
2. ∠ABC + ∠CBF = 90°
3. ∠ABD = ∠ABF + ∠FBD = 90° + ∠FBD
4. 注意到∠CBF + ∠FBD = ∠CBD = 180°（平角）
5. 因此∠ABC + ∠ABD = 180°

**证毕。**

# Prop 14: 邻角和为180°则成直线

**命题：** 若两邻角之和等于两直角，则两角非公共边成一直线。

**已知：** ∠ABC + ∠ABD = 180°，共边AB
**求证：** CBD是直线

**反证法：**
1. 若CBD不是直线，延长CB至E使CBE为直线
2. 则∠ABC + ∠ABE = 180°（Prop 13）
3. 但∠ABC + ∠ABD = 180°（已知）
4. 因此∠ABD = ∠ABE
5. 小角等于大角（除非D在BE上），矛盾
6. 故CBD是直线

**证毕。**

# Prop 16: 三角形外角大于任一内对角

**命题：** 三角形一边延长，外角大于任一内对角。

**已知：** △ABC，延长BC至D
**求证：** ∠ACD > ∠ABC，∠ACD > ∠BAC

**证明（以∠ACD > ∠ABC为例）：**
1. 取AC中点E（Prop 10）
2. 延长BE至F使EF = BE
3. 连接FC
4. △AEB ≅ △CEF (SAS: AE=CE, BE=FE, ∠AEB=∠CEF对顶角)
5. 因此∠BAE = ∠FCE
6. 即∠BAC = ∠FCE
7. 但∠ACD = ∠ACF + ∠FCD > ∠ACF = ∠FCE = ∠BAC
8. 因此∠ACD > ∠BAC

**证毕。**

# Prop 18: 大边对大角

**证毕。**

# Prop 19: 大角对大边

**命题：** 三角形中，大角对大边。

**已知：** △ABC，∠ABC > ∠ACB
**求证：** AC > AB

**反证法：**

3. 若AC < AB，则∠ABC < ∠ACB（Prop 18），矛盾
4. 因此AC > AB

# Prop 20: 三角形两边和大于第三边

**命题：** 三角形两边之和大于第三边。

**已知：** △ABC
**求证：** AB + AC > BC，AB + BC > AC，AC + BC > AB

**证明（证AB + AC > BC）：**
1. 延长BA至D使AD = AC（Prop 3）
2. 连接CD
3. △ACD等腰，故∠ACD = ∠ADC（Prop 5）

5. 在△BCD中，∠BCD > ∠BDC

7. 即BA + AD > BC
8. 故AB + AC > BC

**证毕。**
