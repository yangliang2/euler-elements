# Prop 21: 三角形两边和小于第三边两倍延长

**命题：** 从三角形一边两端点作相交两线段，其和小于三角形另两边之和，夹角更大。

**已知：** △ABC，延长BA至D，延长CA至E，DE交于F
**求证：** BD + CE > BC，∠BFC > ∠BAC

**证明：**
1. △BDF中，BD + DF > BF（Prop 20）
2. △CEF中，CE + EF > CF（Prop 20）
3. 相加：BD + CE + DF + EF > BF + CF
4. 即BD + CE + DE > BF + CF
5. 又BF + CF > BC（Prop 20）
6. 因此BD + CE > BC
7. ∠BFC = ∠FDC + ∠FCD > ∠FCD = ∠ACB（外角大于内对角）
8. 同理∠BFC > ∠ABC，故∠BFC > ∠BAC

**证毕。**

# Prop 22: 作三角形等于三线段和

**命题：** 作三角形，使其三边分别等于已知三线段（任两边和大于第三边）。

**已知：** 三线段a, b, c，任两边和>第三边
**求作：** △ABC使BC=a, AC=b, AB=c

**构造：**
1. 作射线DX
2. 在DX上截取DY = a，YZ = b
3. 以Y为圆心，c为半径作圆
4. 以Z为圆心，b为半径作圆
5. 两圆交于W
6. △YWZ即为所求（YW=c, YZ=a, WZ=b）

**证毕。**

# Prop 23: 作角等于已知角

**命题：** 在已知直线上作一角等于已知角。

**已知：** 直线AB，点A，∠DCE
**求作：** 在A点作∠BAL = ∠DCE

**构造：**

2. 连接DE
3. 在AB上作△AFG ≅ △CDE（Prop 22）
4. 使AF = CD，AG = CE，FG = DE
5. 则∠FAG = ∠DCE

**证毕。**

# Prop 24: 两边夹大角对大边

**命题：** 两三角形两边对应相等，夹角不等，则夹角大的对边大。

**已知：** AB=DE, AC=DF, ∠A > ∠D
**求证：** BC > EF

**证明：**
1. 在∠BAC内作∠BAG = ∠EDF，AG = DF = AC（Prop 23）
2. 则△ABG ≅ △DEF (SAS)，故BG = EF
3. 连接CG
4. △ACG等腰（AG=AC），故∠ACG = ∠AGC
5. ∠BGC = ∠AGC > ∠ACG（整体大于部分）
6. 因此∠BGC > ∠BCG
7. 故BC > BG = EF

**证毕。**

# Prop 25: 大边对夹大角

**命题：** 两三角形两边对应相等，第三边不等，则大边对大角。

**已知：** AB=DE, AC=DF, BC > EF
**求证：** ∠A > ∠D

**反证法：**
1. 设∠A ≤ ∠D
2. 若∠A = ∠D，则BC = EF（Prop 4），矛盾
3. 若∠A < ∠D，则BC < EF（Prop 24），矛盾
4. 因此∠A > ∠D

**证毕。**

# Prop 26: 角边角全等 (ASA)

**命题：** 两三角形两角及夹边对应相等，则全等。

**已知：** ∠B=∠E, ∠C=∠F, BC=EF
**求证：** △ABC ≅ △DEF

**反证法：**
1. 设AB ≠ DE，不妨设AB > DE
2. 在AB上截取BG = DE
3. 则△GBC ≅ △DEF (SAS)
4. 故∠GCB = ∠F = ∠ACB
5. 但∠GCB < ∠ACB（G在AB上），矛盾
6. 因此AB = DE，同理AC = DF
7. △ABC ≅ △DEF (SSS或SAS)

**证毕。**

# Prop 27: 内错角相等则平行

**命题：** 一直线与两直线相交，内错角相等，则两直线平行。

**已知：** EF交AB、CD于E、F，∠AEF = ∠EFD
**求证：** AB ∥ CD

**反证法：**
1. 设AB不平行于CD，延长后交于G
2. △EFG中，∠AEF是外角
3. 则∠AEF = ∠EFG + ∠EGF > ∠EFG
4. 但∠AEF = ∠EFD = ∠EFG（已知），矛盾
5. 因此AB ∥ CD

**证毕。**

# Prop 28: 同位角/同旁内角判定平行

**命题：** 一直线与两直线相交，同位角相等或同旁内角互补，则两直线平行。

**情况1：同位角相等**
1. ∠EGB = ∠GHD（已知同位角相等）
2. ∠EGB = ∠AGH（对顶角，Prop 15）
3. 因此∠AGH = ∠GHD（内错角相等）
4. 故AB ∥ CD（Prop 27）

**情况2：同旁内角互补**
1. ∠BGH + ∠GHD = 180°（已知）
2. ∠BGH + ∠AGH = 180°（邻补角）
3. 因此∠AGH = ∠GHD（内错角相等）
4. 故AB ∥ CD（Prop 27）

**证毕。**

# Prop 29: 平行线性质

**命题：** 一直线与两平行线相交，内错角相等，同位角相等，同旁内角互补。

**已知：** AB ∥ CD，EF交于E、F
**求证：**
(1) ∠AEF = ∠EFD（内错角）
(2) ∠CEF = ∠EFD（同位角）
(3) ∠BEF + ∠EFD = 180°（同旁内角）

**证明(1)：**
1. 设∠AEF ≠ ∠EFD，不妨设∠AEF > ∠EFD
2. 则内错角不等，AB不平行于CD（Prop 27逆否）
3. 矛盾，故∠AEF = ∠EFD

**证明(2)：**
1. ∠AEF = ∠CEF'（对顶角）
2. 但∠AEF = ∠EFD（已证）
3. 因此∠CEF = ∠EFD

**证明(3)：**
1. ∠BEF + ∠AEF = 180°（邻补角）
2. ∠AEF = ∠EFD（已证）
3. 因此∠BEF + ∠EFD = 180°

**证毕。**

# Prop 30: 平行于同一直线的两直线平行

**命题：** 两直线都平行于第三直线，则它们互相平行。

**已知：** AB ∥ EF，CD ∥ EF
**求证：** AB ∥ CD

**证明：**
1. 作直线GK交三直线
2. ∠AGK = ∠GHF（AB ∥ EF，同位角，Prop 29）
3. ∠GHF = ∠GKD（CD ∥ EF，同位角，Prop 29）
4. 因此∠AGK = ∠GKD（内错角相等）
5. 故AB ∥ CD（Prop 27）

**证毕。**
