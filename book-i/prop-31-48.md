# Prop 33: 平行四边形对边相等

# Prop 47: 毕达哥拉斯定理

**命题：** 直角三角形斜边上正方形等于两直角边上正方形之和。

**已知：** △ABC，∠A = 90°
**求证：** BC² = AB² + AC²

**证明（欧几里得经典证法）：**
1. 在BC上作正方形BDEC
2. 在AB、AC上作正方形ABFG、ACKH
3. 过A作AL ∥ BD交BC于L
4. ▱BDLM = 2△ABD（Prop 41）
5. 正方形ABFG = 2△FBC（同底FB，同高，Prop 41）
6. △ABD ≅ △FBC (SAS: AB=FB, BD=BC, ∠ABD=∠FBC)
7. 因此▱BDLM = 正方形ABFG
8. 同理▱CELM = 正方形ACKH
9. 正方形BDEC = ▱BDLM + ▱CELM = AB² + AC²

**证毕。**
（数学史上最著名定理之一）

# Prop 48: 毕达哥拉斯逆定理

**命题：** 三角形一边上正方形等于另两边上正方形之和，则夹角为直角。

**已知：** △ABC，BC² = AB² + AC²
**求证：** ∠A = 90°

**证明：**
1. 在A作AD ⊥ AB，AD = AC（Prop 11, Prop 3）
2. 连接BD
3. △ABD为直角三角形
4. BD² = AB² + AD² = AB² + AC²（Prop 47）
5. 因此BD² = BC²，BD = BC
6. △ABC ≅ △ABD (SSS)
7. 因此∠BAC = ∠BAD = 90°

**证毕。**
