# Book II: 几何代数

**主题：** 用几何方法处理代数问题（面积变换）

---

# Prop 1: 两线段和乘一截线

**命题：** 两线段之一被分成若干部分，两线段构成的矩形等于各部分与未分线段构成的矩形之和。

**代数表达：** a(b+c+d) = ab + ac + ad

**几何表达：** 若AB = a，BC被分为BD, DE, EC
则AB·BC = AB·BD + AB·DE + AB·EC

**证毕。**

---

# Prop 2: 线段整体乘部分

**命题：** 若一线段被任意分成两部分，该线段与其中一部分构成的矩形，加上另一部分上正方形，等于整体上的正方形。

**代数表达：** a² = a(a-b) + b² + (a-b)b ... 简化：a² = a(a-b) + b(a-b+b) = a(a-b) + ab

**标准形式：** (a+b)a + b² = (a+b)² - ab ... 更简洁：a·b + a² = a(a+b)

**几何：** 线段AB = a，分为AC = b，CB = a-b
则AB·AC + CB² = AB² - ... 修正：AB·AC + CB·AB = AB²

**正确表述：** 整体×部分 + 另一部分×整体 = 整体²
即 a·b + a·(a-b) = a²

**证毕。**

---

# Prop 3: 线段与差构成的矩形

**命题：** 若一线段被任意分成两部分，该线段与其中一部分构成的矩形，等于部分上正方形加上两部分构成的矩形。

**代数：** a·b = b² + b(a-b)

**几何：** AB = a，分为AC = b，CB = a-b
AB·AC = AC² + AC·CB

**证毕。**

---

# Prop 4: 整体平方公式

**命题：** 若一线段被任意分成两部分，整体上的正方形等于部分上正方形之和，加上两部分构成的矩形的两倍。

**代数：** (a+b)² = a² + b² + 2ab

**几何：** AB = a+b，分为AC = a，CB = b
AB² = AC² + CB² + 2·AC·CB

**证毕。**
（完全平方公式几何证明）

---

# Prop 5: 和差公式

**命题：** 若一线段被分成相等和不等两部分，不等部分构成的矩形，加上两分点间线段上正方形，等于线段一半上正方形。

**设：** AB = 2a，等分于C，不等分于D，AD = a+b，DB = a-b，CD = b
**代数：** (a+b)(a-b) + b² = a²
即 a² - b² + b² = a²

**几何：** AD·DB + CD² = AC²

**证毕。**
（平方差公式变形）

---

# Prop 6: 另一种和差公式

**命题：** 若一线段被平分，再延长，则整体与延长部分构成的矩形，加上一半上正方形，等于一半与延长之和上正方形。

**设：** AB = 2a，平分于C，延长BD = b
**代数：** (2a+b)b + a² = (a+b)²
即 2ab + b² + a² = a² + 2ab + b² ✓

**几何：** AD·DB + CB² = CD²

**证毕。**

---

# Prop 7: 和的平方与部分平方

**命题：** 若一线段被任意分成两部分，整体与一部分之和上正方形，等于整体上正方形加上该部分上正方形，加上整体与该部分构成的矩形的两倍。

**设：** AB = a，分为AC = b，CB = a-b
整体+部分 = a+b（延长）
**代数：** (a+b)² = a² + b² + 2ab

**几何：** (AB+AC)² = AB² + AC² + 2·AB·AC

**证毕。**
（与Prop 4等价）

---

# Prop 8: 四倍矩形加一平方

**命题：** 若一线段被任意分成两部分，四倍整体与一部分构成的矩形，加上另一部分上正方形，等于整体与前一部分之和上正方形。

**设：** AB = a，分为AC = b
**代数：** 4ab + (a-b)² = (a+b)²
4ab + a² - 2ab + b² = a² + 2ab + b² ✓

**几何：** 4·AB·AC + CB² = (AB+AC)²

**证毕。**

---

# Prop 9: 和方加差方等于二倍和

**命题：** 若一线段被分成相等和不等两部分，不等部分上正方形之和，等于等分上正方形与两分点间线段上正方形之和的两倍。

**设：** AB = 2a，等分于C，不等分于D
AD = a+b，DB = a-b
**代数：** (a+b)² + (a-b)² = 2(a² + b²)
a²+2ab+b² + a²-2ab+b² = 2a² + 2b² ✓

**几何：** AD² + DB² = 2(AC² + CD²)

**证毕。**

---

# Prop 10: 和差关系

**命题：** 若一线段被平分后再延长，整体与延长部分上正方形之和，等于一半上正方形与一半加延长之和上正方形之和的两倍。

**设：** AB = 2a，平分于C，延长BD = b
AD = 2a+b，DB = b
**代数：** (2a+b)² + b² = 2(a² + (a+b)²)
4a²+4ab+b² + b² = 2a² + 2(a²+2ab+b²) = 4a²+4ab+2b² ✓

**几何：** AD² + DB² = 2(AC² + CD²)

**证毕。**

---

# Prop 11: 分割线段使矩形等于正方形

**命题：** 分割已知线段，使整体与一部分构成的矩形等于另一部分上正方形。

**已知：** AB
**求作：** 分于C使AB·BC = AC²

**代数：** 设AB = 1，AC = x，则BC = 1-x
1·(1-x) = x²
x² + x - 1 = 0
x = (√5 - 1)/2 ≈ 0.618（黄金分割！）

**几何构造：**
1. 在AB上作正方形ABDE
2. 取AD中点F
3. 延长FA至G使FG = FB
4. 在AB上截取AH = AG
5. H即为所求分点

**证毕。**
（黄金分割的欧几里得构造）

# Prop 12: 钝角三角形余弦定理原型

**命题：** 钝角三角形中，钝角对边上正方形，等于另两边上正方形之和，加上一边与另一边在其上投影构成的矩形的两倍。

**已知：** △ABC，∠A钝角，BD ⊥ CA延长线
**求证：** BC² = AB² + AC² + 2·AC·AD

**证明：**
1. CD = CA + AD
2. BC² = BD² + CD² = BD² + (CA+AD)²
3. = BD² + CA² + 2·CA·AD + AD²
4. = (BD²+AD²) + CA² + 2·CA·AD
5. = AB² + AC² + 2·AC·AD

**证毕。**
（余弦定理：c² = a² + b² + 2ab·cos(180°-A) = a² + b² - 2ab·cosA）

---

# Prop 13: 锐角三角形余弦定理原型

**命题：** 锐角三角形中，锐角对边上正方形，等于另两边上正方形之和，减去一边与另一边在其上投影构成的矩形的两倍。

**已知：** △ABC，∠A锐角，BD ⊥ AC
**求证：** BC² = AB² + AC² - 2·AC·AD

**证明：**
1. CD = AC - AD
2. BC² = BD² + CD² = BD² + (AC-AD)²
3. = BD² + AC² - 2·AC·AD + AD²
4. = (BD²+AD²) + AC² - 2·AC·AD
5. = AB² + AC² - 2·AC·AD

**证毕。**
（余弦定理：c² = a² + b² - 2ab·cosA）

---

# Prop 14: 作正方形等于已知多边形

**命题：** 作一个正方形等于已知多边形。

**已知：** 多边形A
**求作：** 正方形 = A

**构造：**
1. 作平行四边形BCDE = A（Prop 45，第一卷）
2. 若BCDE是矩形且邻边相等，即为正方形
3. 若不是，设BE > BC
4. 延长BC至F使CF = BE
5. 平分BF于G
6. 以G为圆心，GB为半径作半圆
7. 延长EC交半圆于H
8. 则CH² = BE·BC（由Prop 13，直角三角形CHB）
9. 在CH上作正方形CHKL

**证毕。**
（化圆为方问题的前置：化多边形为方）
