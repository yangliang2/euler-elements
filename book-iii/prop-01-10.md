# Book III: 圆的性质

**主题：** 圆、弦、切线、圆内接图形

---

# Prop 1: 找圆心

**命题：** 找已知圆的圆心。

**已知：** 圆ABC
**求作：** 圆心

**构造：**
1. 在圆上任取弦AB
2. 作AB中点D（Prop 10，第一卷）
3. 过D作DC ⊥ AB，交圆于C
4. 作CB中点E
5. E即为圆心

**证明：**
1. 假设圆心不在E，设圆心为F
2. 连接FA, FB, FC
3. FA = FB = FC（半径）
4. △FAD ≅ △FBD (SSS或SAS)，故∠FDA = ∠FDB = 90°
5. 因此FD ⊥ AB
6. 但CD ⊥ AB，且过D只有一条垂线
7. 因此F在CD上
8. 又FC = FB，F在CB中垂线上
9. E是CB中点，故F = E

**证毕。**

---

# Prop 2: 圆心必在弦中垂线上

**命题：** 若圆上两点连线，则圆心必在该线段中垂线上。

**已知：** 圆ABC，弦AB
**求证：** 圆心在AB中垂线上

**证明：**
1. 设圆心为O
2. OA = OB（半径）
3. 因此O在AB中垂线上（到两端点等距的点在中垂线上）

**证毕。**

---

# Prop 3: 圆心分弦性质

**命题：** 过圆心的直线若平分不过圆心的弦，则垂直于该弦；若垂直，则平分。

**已知：** 圆ABC，弦CD不过圆心，直线AB过圆心O
**情况1：** AB平分CD于F
**求证：** AB ⊥ CD

**证明：**
1. 连接OC, OD
2. OC = OD（半径）
3. CF = FD（已知）
4. OF公共
5. △OCF ≅ △ODF (SSS)
6. 因此∠OFC = ∠OFD = 90°

**情况2：** AB ⊥ CD于F
**求证：** CF = FD

**证明：**
1. OC = OD（半径）
2. OF公共
3. ∠OFC = ∠OFD = 90°（已知）
4. △OCF ≅ △ODF (SAS)
5. 因此CF = FD

**证毕。**

---

# Prop 4: 不过圆心的相交弦不等分

**命题：** 圆内不过圆心的两弦相交，则它们不能互相平分。

**已知：** 圆ABC，弦AB、CD交于E，都不过圆心
**求证：** AB、CD不能互相平分

**反证法：**
1. 设AE = EB，CE = ED
2. 由Prop 3，AB中垂线过圆心
3. CD中垂线也过圆心
4. 两弦中垂线都过圆心且都过E
5. 因此E是圆心
6. 矛盾（已知弦不过圆心）

**证毕。**

---

# Prop 5: 相交圆不同心

**命题：** 两圆相交，则不同心。

**已知：** 圆ABC、ABD相交于A、B
**求证：** 两圆不同心

**反证法：**
1. 设两圆同心，圆心为E
2. 连接EA、EB
3. EA = EB（圆ABC半径）
4. EA = EB（圆ABD半径）
5. 过E作AB中垂线交AB于F
6. 在圆ABC中，EF ⊥ AB，故AF = FB（Prop 3）
7. 在圆ABD中，同理AF = FB
8. 两圆有相同圆心和中垂线
9. 但A、B固定，两圆重合
10. 矛盾（相交圆不重合）

**证毕。**

---

# Prop 6: 相切圆不同心

**命题：** 两圆相切，则不同心。

**已知：** 圆ABC、CDE相切于C
**求证：** 两圆不同心

**反证法：**
1. 设两圆同心，圆心为F
2. FC是两圆公共半径
3. 在圆ABC上任取B，连接FB
4. FB = FC（半径）
5. 在圆CDE上任取D，连接FD
6. FD = FC（半径）
7. 因此FB = FD = FC
8. B、C、D都在以F为圆心同一圆上
9. 两圆重合
10. 矛盾

**证毕。**

---

# Prop 7: 过圆心到弦端点的线段最大

**命题：** 圆心到圆上一点的连线，若延长交圆于另一点，则靠近圆心的一半最大。

**已知：** 圆ABCD，直径AD，圆心F，弦BC过F
**求证：** FA = FD > FB = FC > 其他

**证明：**
1. FA = FD（半径）
2. 对弦BC上任意点E（非端点），连接FE并延长交圆于G
3. FE < FG（F在圆内，E在FG之间）
4. 但FG = FA（半径）
5. 因此FA > FE
6. 当E为B或C时，FE = FB = FC

**证毕。**

---

# Prop 8: 圆外一点到圆的线段

**命题：** 圆外一点到圆心连线，穿过圆的部分最大，延长部分最小。

**已知：** 圆ABC，圆心D，圆外点E
**求证：**
- EA最大（E过D交圆最近点）
- EF最小（E过D交圆最远点）

**证明：**
1. 连接ED，交圆于A、F（A近，F远）
2. 在圆上任取B，连接EB、DB
3. EB + BD > ED = EA + AD（三角形不等式）
4. BD = AD（半径）
5. 因此EB > EA
6. 同理可证EF最小

**证毕。**

---

# Prop 9: 圆内点到圆周三点连线相等则圆心

**命题：** 圆内一点到圆周三点连线相等，则该点为圆心。

**已知：** 圆ABC，点D，DA = DB = DC
**求证：** D是圆心

**证明：**
1. 假设D不是圆心，设圆心为E
2. 连接EA, EB, EC
3. EA = EB = EC（半径）
4. 在△EAD和△EBD中，DA = DB，EA = EB，ED公共
5. △EAD ≅ △EBD (SSS)
6. 因此∠ADE = ∠BDE
7. 同理∠ADE = ∠CDE
8. 因此A、D、B、C共线或D = E
9. 若共线，则A、B、C在一直径上，但DA = DB = DC不可能
10. 因此D = E，D是圆心

**证毕。**

---

# Prop 10: 两圆相交最多两点

**命题：** 两圆相交，交点不多于两个。

**反证法：**
1. 设两圆交于A、B、C三点
2. 找两圆圆心（Prop 1）
3. 两圆圆心都在AB、BC、CA的中垂线上
4. 但三点确定唯一圆
5. 因此两圆重合，矛盾

**证毕。**
