# Prop 3: 圆内接给定正方形

**命题：** 在已知圆内作内接正方形。

**已知：** 圆ABCD
**求作：** 内接正方形

**构造：**
1. 作两垂直直径AC、BD
2. 连接AB、BC、CD、DA
3. ABCD即为正方形

**证明：**
1. AC = BD，互相平分且垂直
2. AB = BC = CD = DA（全等直角三角形斜边）
3. ∠ABC = 90°（半圆上的角）

**证毕。**

# Prop 7: 圆内接给定正六边形

**命题：** 在已知圆内作内接正六边形。

**构造：**
1. 以圆上一点A为圆心，半径为圆半径作弧交圆于B
2. △OAB为等边三角形（O为圆心）
3. 圆心角∠AOB = 60°
4. 连续截取，得六顶点
5. 连接成正六边形

**证毕。**
（边长=半径，最简单的正多边形作图）

# Prop 9: 圆内接给定正十五边形

**命题：** 在已知圆内作内接正十五边形。

**关键：** 360°/15 = 24°
**方法：**
1. 正六边形圆心角 = 60°
2. 正十边形圆心角 = 36°
3. 60° - 36° = 24° = 360°/15
4. 作正六边形和正十边形
5. 利用角度差确定正十五边形顶点

**证毕。**
