# Book V: 比例论

**主题：** 欧多克索斯比例理论（适用于可公度和不可公度量）

**核心概念：** 
- 比（ratio）：两同类量之间的关系
- 比例（proportion）：两个比相等 a:b = c:d

---

# Prop 1: 倍量比较

**命题：** 若若干量分别是若干等量的同倍量，则无论前者个数多少，前者之和也是后者之和的同倍量。

**代数：** n·a + n·b + ... = n·(a + b + ...)

**几何：** 若MA、MB分别是A、B的M倍量，则MA + MB = M(A + B)

**证毕。**

---

# Prop 2: 倍量的倍量

**命题：** 若第一量是第二量的倍量，第三量是第四量的倍量，倍数相同；且第五量是第二量的倍量，第六量是第四量的倍量，倍数相同；则第一量与第五量之和是第二量的倍量，第三量与第六量之和是第四量的倍量，倍数相同。

**代数：** 
- a = n·b, c = n·d
- e = m·b, f = m·d
- 则 a + e = (n+m)·b, c + f = (n+m)·d

**证毕。**

---

# Prop 3: 倍量比较（扩展）

**命题：** 类似Prop 1，针对多个量的情况。

**证毕。**

---

# Prop 4: 等比取等倍仍等比

**命题：** 若第一量比第二量等于第三量比第四量，取第一量与第三量的同倍量，以及第二量与第四量的同倍量，则倍量之比仍相等。

**代数：** 
- a:b = c:d
- 则 m·a : m·b = n·c : n·d = a:b = c:d

**证毕。**

---

# Prop 5: 大比减小比

**命题：** 若第一量是第二量的倍量，第三量是第四量的倍量，倍数相同；且第六量是第四量的倍量，第五量是第二量的倍量，倍数相同；则第一量与第五量之和是第二量的倍量，第三量与第六量之和是第四量的倍量，倍数相同。

**代数：** 
- a = n·b, c = n·d
- e = m·b, f = m·d
- 则 a + e = (n+m)·b, c + f = (n+m)·d

**证毕。**

---

# Prop 6: 等比的传递性（倍量）

**命题：** 若两量分别是另两量的同倍量，且从前两者中减去后两者的同倍量，则余量要么相等，要么分别是后两者的同倍量。

**代数：** 
- a = n·b, c = n·d
- 减去 e = m·b, f = m·d
- 则 a-e = (n-m)·b, c-f = (n-m)·d

**证毕。**

---

# Prop 7: 等比量相加

**命题：** 相等量与同一量之比相等；且等量之比等于同一量之比。

**代数：** 
- 若 a = b，则 a:c = b:c 且 c:a = c:b

**证毕。**

---

# Prop 8: 不等量与同一量之比

**命题：** 不等量与同一量之比，大者比大。

**代数：** 
- 若 a > b，则 a:c > b:c
- 且 c:a < c:b

**证毕。**

---

# Prop 9: 与等量之比相等则量相等

**命题：** 与同一量之比相等的量彼此相等；且同一量与比相等的量之比也相等。

**代数：** 
- 若 a:c = b:c，则 a = b
- 若 c:a = c:b，则 a = b

**证毕。**

---

# Prop 10: 与不等量之比相等则量不等

**命题：** 与不等量之比相等的量彼此不等；且同一量与比相等的量之比也不等。

**证毕。**

---

# Prop 11: 等比的传递性

**命题：** 若 a:b = c:d 且 c:d = e:f，则 a:b = e:f

**证毕。**
（比例关系的传递性）

---

# Prop 12: 多比连等

**命题：** 若多个比都相等，则前项之和比后项之和等于其中任一比。

**代数：** 
- 若 a:b = c:d = e:f
- 则 (a+c+e):(b+d+f) = a:b

**证毕。**
（平均值的理论基础）

---

# Prop 13: 比的不等传递

**命题：** 若 a:b > c:d 且 c:d > e:f，则 a:b > e:f

**证毕。**

---

# Prop 14: 合比定理

**命题：** 若 a:b = c:d 且 a > c，则 b > d

**证毕。**

---

# Prop 15: 部分与整体之比

**命题：** 部分与整体之比等于同类的部分与整体之比。

**证毕。**

---

# Prop 16: 更比定理

**命题：** 若四个量成比例，则更比也成比例。

**代数：** 
- 若 a:b = c:d，则 a:c = b:d

**证毕。**
（比例基本性质之一）

---

# Prop 17: 合分比定理

**命题：** 若 a:b = c:d，则 (a+b):b = (c+d):d

**证毕。**
（合比定理）

---

# Prop 18: 分比定理

**命题：** 若 a:b = c:d，则 (a-b):b = (c-d):d（设 a>b, c>d）

**证毕。**

---

# Prop 19: 合分比逆定理

**命题：** 若 (a+b):(c+d) = a:c = b:d，则原四量成比例。

**证毕。**

---

# Prop 20: 多比排序

**命题：** 若多个比各相等，且前项之和大于后项之和，则单个前项也大于对应后项。

**证毕。**

---

# Prop 21: 有公度的量之比

**命题：** 有公度的量之比等于数之比。

**说明：** 这是可公度量的特征，可以表示为有理数比。

**证毕。**

---

# Prop 22: 多比相等的一般形式

**命题：** 若任意多个量成比例，则前项中任取一些之和比后项中对应一些之和等于整个前项之和比整个后项之和。

**证毕。**

---

# Prop 23: 等比的连乘

**命题：** 若 a:b = e:f 且 c:d = e:f，则 (a·c):(b·d) = e:f

**证毕。**

---

# Prop 24: 复合比

**命题：** 若 a:c = d:f 且 c:b = e:d，则 a:b = e:f

**证毕。**
（比的复合运算）

# Prop 25: 四量成比例的不等关系

**代数：** 
- 若 a:b = c:d 且 a > c，则 b > d
