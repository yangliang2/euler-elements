# Prop 1: 等高的三角形/平行四边形面积比等于底边比

**命题：** 等高的三角形或平行四边形之比等于底边之比。

**已知：** △ABC和△ACD同高（从A到BD的垂线）
**求证：** △ABC:△ACD = BC:CD

**证明：**
1. 延长BD至H、L使BH = n·BC，DL = m·CD
2. △ABH = n·△ABC，△ADL = m·△ACD
3. 由第五卷比例理论，面积比等于底边比

**证毕。**

---

# Prop 2: 平行于三角形一边的直线

**命题：** 平行于三角形一边的直线截另外两边成比例线段。

**已知：** △ABC，DE ∥ BC，D在AB上，E在AC上
**求证：** AD:DB = AE:EC

**证明：**
1. 连接BE、CD
2. △BDE = △CDE（同底DE，同高，因DE∥BC）
3. △ADE与△BDE同高，面积比 = AD:DB
4. △ADE与△CDE同高，面积比 = AE:EC
5. 因此AD:DB = AE:EC

**证毕。**
（基本比例定理/泰勒斯定理）

# Prop 3: 角平分线定理

**命题：** 三角形角平分线分对边成邻边之比。

**已知：** △ABC，AD平分∠A交BC于D
**求证：** BD:DC = AB:AC

**证明：**
1. 过C作CE ∥ AD交BA延长线于E
2. ∠BAD = ∠AEC（同位角）
3. ∠DAC = ∠ACE（内错角）
4. ∠BAD = ∠DAC，因此∠AEC = ∠ACE
5. △ACE等腰，AE = AC
6. BD:DC = BA:AE = BA:AC（Prop 2）

**证毕。**

# Prop 4: 等角三角形相似

**命题：** 两三角形两角对应相等，则相似（对应边成比例，夹角相等）。

**已知：** △ABC和△DEF，∠A = ∠D，∠B = ∠E
**求证：** △ABC ~ △DEF，AB:DE = BC:EF = AC:DF

**证明：**
1. ∠C = ∠F（三角形内角和）
2. 在AB、AC上取AG = DE，AH = DF
3. △AGH ≅ △DEF（SAS）
4. ∠AGH = ∠B = ∠ABC，因此GH ∥ BC
5. AG:AB = AH:AC（Prop 2）
6. 即DE:AB = DF:AC
7. 同理可证其他边比例

**证毕。**
（AA相似判定）

# Prop 5: 两边成比例且夹角相等则相似

**命题：** 两三角形两边成比例且夹角相等，则相似。

**已知：** △ABC和△DEF，AB:DE = AC:DF，∠A = ∠D
**求证：** △ABC ~ △DEF

**证明：**
1. 在AB、AC上取AG = DE，AH = DF
2. AG:AB = AH:AC（由已知比例）
3. 因此GH ∥ BC（Prop 2逆）
4. ∠AGH = ∠B，∠AHG = ∠C
5. △AGH ≅ △DEF（SAS），因此∠B = ∠E，∠C = ∠F
6. 由Prop 4，三角形相似

**证毕。**
（SAS相似判定）

# Prop 6: 三边成比例则相似

**命题：** 两三角形三边成比例，则相似。

**已知：** △ABC和△DEF，AB:DE = BC:EF = AC:DF
**求证：** △ABC ~ △DEF

**证明：**
1. 在AB、AC上取AG = DE，AH = DF
2. 由比例关系，GH ∥ BC
3. ∠AGH = ∠B，∠AHG = ∠C
4. △AGH ≅ △DEF（SSS）
5. 因此∠B = ∠E，∠C = ∠F
6. 由Prop 4，三角形相似

**证毕。**
（SSS相似判定）

# Prop 8: 直角三角形相似

**命题：** 直角三角形斜边和一直角边与另一直角三角形对应成比例，则相似。

**已知：** Rt△ABC和Rt△DEF，∠C = ∠F = 90°，AB:DE = AC:DF
**求证：** △ABC ~ △DEF

**证明：**
1. 勾股定理：BC² = AB² - AC²，EF² = DE² - DF²
2. 由比例关系，设AB = k·DE，AC = k·DF
3. BC² = k²(DE² - DF²) = k²·EF²
4. 因此BC = k·EF
5. 三边成比例，由Prop 6，三角形相似

**证毕。**

# Prop 9: 截取比例线段

**命题：** 从给定直线上截取给定比例的部分。

**已知：** 线段AB，比例C:D
**求作：** 在AB上截取AE使AE:EB = C:D

**构造：**
1. 从A作任意射线
2. 在射线上取AF = C，FG = D
3. 连接GB
4. 过F作FH ∥ GB交AB于H
5. AH:HB = AF:FG = C:D（Prop 2）
6. H即为所求点

**证毕。**

# Prop 10: 截取相似三角形

**命题：** 作给定三角形的相似三角形，使其一边等于给定线段。

**已知：** △ABC，线段D
**求作：** △DEF ~ △ABC，且EF = D

**构造：**
1. 在BC延长线上取CG = D
2. 过G作GH ∥ AB交AC延长线于H
3. 过H作HF ∥ BC交AB延长线于F
4. △FGH ~ △ABC，且GH = D

**证毕。**
