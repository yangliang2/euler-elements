# 定义

- **单位：** 度量每个存在的事物，称为一。
- **数：** 由单位组成的 multitude。
- **部分：** 较小数量度较大量。
- **部分数：** 较小数量较大量的部分。
- **偶数：** 能被2整除的数。
- **奇数：** 不能被2整除的数（或比偶数多一单位）。
- **素数：** 只能被单位整除的数。
- **合数：** 能被某数量度的数。
- **互素：** 只有单位能量度的数。
- **比例数：** 有公度的数（可表示为整数比）。

# Prop 2: 求最大公约数

**命题：** 给定两不互素的数，求它们的最大公约数。

**欧几里得算法：**
1. 设两数为AB、CD，AB > CD
2. 若CD能量AB，则CD是公约数
3. 若不能，用CD从AB中连续减去，得余数EF < CD
4. 若EF能量CD，则EF是公约数
5. 若不能，重复步骤
6. 最终余数为单位（互素）或找到公约数

**证毕。**
（欧几里得算法——历史上最重要的算法之一）

# Prop 20: 素数无穷多

**命题：** 素数的个数比任何给定的素数集合都多。

**证明（反证法）：**
1. 设素数只有有限个：p₁, p₂, ..., pₙ
2. 构造数 N = p₁·p₂·...·pₙ + 1
3. N不被任何pᵢ整除（余数为1）
4. 因此N是素数或含有新素因子
5. 矛盾，因此素数无穷多

**证毕。**
（数学史上最优美证明之一）

# Prop 31: 合数的素因子

**命题：** 任何合数都能被某素数整除。

**证明：**
1. 设A是合数
2. 则某数B量A
3. 若B是素数，证毕
4. 若B是合数，则某数C量B
5. C量B且B量A，因此C量A
6. 重复此过程，最终找到素因子
（因为数不断减小，必终止于素数）

**证毕。**
（算术基本定理的基础）
