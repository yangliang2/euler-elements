# Prop 2: 圆之比等于直径平方比

**命题：** 两圆之比等于直径平方比。

**穷竭法证明：**
1. 设两圆为A、B，直径分别为d₁、d₂
2. 假设A:B ≠ d₁²:d₂²
3. 在圆A内作内接正多边形，使其与圆A之差小于任意给定量
4. 在圆B内作相似正多边形
5. 多边形之比等于d₁²:d₂²（Prop 1）
6. 若A:B > d₁²:d₂²，则存在内接多边形使多边形:B > d₁²:d₂²
7. 但多边形:B = 多边形:多边形B = d₁²:d₂²，矛盾
8. 同理A:B < d₁²:d₂²也矛盾
9. 因此A:B = d₁²:d₂²

**证毕。**
（πr² = π(d/2)² = (π/4)d²，故圆面积比等于直径平方比）

# Prop 10: 圆锥与圆柱的体积

**命题：** 圆锥是等底等高圆柱的三分之一。

**穷竭法证明：**
1. 设圆柱为C，圆锥为V
2. 假设V ≠ C/3
3. 若V > C/3，在圆锥内作内接棱锥
4. 棱锥 > C/3，但棱锥 < V，矛盾
5. 若V < C/3，在圆柱内作外接棱柱
6. 通过穷竭，证明V = C/3

**证毕。**
（V = (1/3)πr²h）

# Prop 18: 球的体积

**命题：** 球之比等于直径立方比。

**穷竭法证明：**
1. 设两球S₁、S₂，直径d₁、d₂
2. 假设S₁:S₂ ≠ d₁³:d₂³
3. 在球内作内接多面体
4. 多面体之比趋近d₁³:d₂³
5. 通过穷竭法证明S₁:S₂ = d₁³:d₂³

**证毕。**
（V = (4/3)πr³ = (π/6)d³）
