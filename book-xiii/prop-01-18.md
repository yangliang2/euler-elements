# Prop 1: 线段中项的构造

**命题：** 将线段分成中外比（黄金分割）。

**构造（回顾第二卷Prop 11）：**
1. 设线段AB
2. 作正方形ABCD
3. 取AD中点E
4. 延长EA至F使EF = EB
5. 在AB上截取AG = AF
6. G将AB分成黄金比：AB:AG = AG:GB

**证毕。**
（黄金分割比 φ = (1+√5)/2 ≈ 1.618）

# Prop 6: 正方形的边与对角线

**命题：** 正方形边与其对角线不可公度。

**证明：**
1. 设正方形边长为1，对角线为√2
2. 假设√2 = p/q（最简分数）
3. 则2 = p²/q²，p² = 2q²
4. p²为偶数，故p为偶数，设p=2r
5. 4r² = 2q²，q² = 2r²
6. q²为偶数，故q为偶数
7. p、q均为偶数，与最简分数矛盾
8. 因此√2无理，边与对角线不可公度

**证毕。**
（无理数经典证明）

# Prop 15: 立方体的作图

**命题：** 作立方体并求其内切于已知球时的边长。

**构造：**
1. 设球的直径为d
2. 立方体对角线 = 边长×√3 = d
3. 因此边长 = d/√3 = d√3/3

**证毕。**
（立方体——最简单的正多面体）

# Prop 16: 正四面体（棱锥）的作图

**命题：** 作正四面体并求其内切于已知球时的边长。

**构造：**
1. 球的直径d与正四面体高的关系
2. 正四面体高h = √(2/3)·a（a为边长）
3. 外接球半径R = (√6/4)a
4. 因此a = (2√6/3)R = (√6/3)d

**证毕。**

# Prop 18: 不存在其他正多面体

**命题：** 除上述五种外，不存在其他正多面体。

**证明：**

**正多面体的条件：**
1. 各面为全等的正多边形
2. 各顶点有相同数量的面相交
3. 每个顶点处的面角之和 < 360°

**分析各种正多边形：**

| 正多边形 | 内角 | 可围成立体角的个数 |
|---------|------|------------------|
| 正三角形 | 60° | 3、4、5个（180°、240°、300°）|
| 正方形 | 90° | 3个（270°）|
| 正五边形 | 108° | 3个（324°）|
| 正六边形 | 120° | 3个=360°（平铺，不能）|
| n≥7边形 | >128.6° | 3个>360°（不能）|

**因此只有五种可能：**
1. **正四面体：** 4个正三角形面，每顶点3面
2. **正八面体：** 8个正三角形面，每顶点4面
3. **正二十面体：** 20个正三角形面，每顶点5面
4. **立方体：** 6个正方形面，每顶点3面
5. **正十二面体：** 12个正五边形面，每顶点3面

**不存在第六种正多面体。**

**证毕。**
